### 7.1 获取键盘输入的两种方式
首先先介绍以下main方法签名的含义:

``` java
public static void main(String[] args){

}
```

这是普遍的main方法的写法，这个写法其实意义是：
- JVM需要读取main方法，所有它是public的
- main方法不需要通过实例来调用，而是属于类的，所以是static
- JVM调用main方法不需要任何返回值，所以是void的
